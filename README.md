# 🚀 Space War V1.0

A 2D space shooting game built with HTML5 Canvas and JavaScript.

---

## 🎮 Controls

- Move Left : A / ←
- Move Right : D / →
- Shoot : Space

---

## ✨ Features

- Player spaceship
- Enemy spaceships
- Score system
- HP system
- Game Over screen

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Canvas API
